# Electron Basic Setup

## Requirements:
* Electron
* Code Editor


## Developement:

### Install Packages
<pre>
npm install
</pre>

### Starting app
<pre>
npm start
</pre>


## Building to ".exe"
electron-packager ./ App-Name --platform=win32 --arch=ia32
[API docu](https://github.com/electron-userland/electron-packager/blob/master/docs/api.md#platform)
